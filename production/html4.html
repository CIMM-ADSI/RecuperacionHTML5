<html> Fecha: 12/05/2016

Un documento HTML 5 simple:

Ejemplo

<!DOCTYPE HTML>
<html>
<head>
<title>Titulo del documento</title>
   
<body>
El contenido de documento
</body>    
</html>

Intenatalo tu mismo

Definición y Uso.

La etiqueta <html> representa la raíz de un documento HTML.

La etiqueta <html> es el contenedor de todos los demás elementos HTML (a excepción de <! DOCTYPE> etiqueta).

La etiqueta <html> indica al navegador que se trata de un documento HTML.

Diferencias entre HTML 4.01 y HTML5.

HTML5 ha añadido un nuevo atributo: manifiesto

Diferencias entre HTML y XHTML

El atributo xmlns se requiere en XHTML, pero no es válido en HTML.

Sin embargo, el validador HTML en http://w3.org no se queja cuando el atributo xmlns no se encuentra en un documento XHTML. Esto se debe a que el espacio de nombres xmlns = "http: //www.w3.org/1999/xhtml" es por defecto, y se añadirá a la etiqueta <html>, incluso si no es necesario incluirla.

Ajustes por defecto CSS.

La mayoría de los navegadores mostrarán el elemento <html> con los siguientes valores predeterminados:

html {
  display: block;
}
html:focus {
  outline: none;
}
