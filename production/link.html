Etiqueta <Link>. Fecha: 12/05/2016

Enlace a una hoja de estilos externa:

Ejemplo

<head> <link rel="stylesheet" type="text/css" href="theme.css"> </head>

Definición y Uso.

La etiqueta <link> define una relacion entre un documento y un recurso externo.

La etiqueta <link> se utiliza para vincular a las hojas de estilo externas

Soporte de Navegadores.

Elemento
<link> Si Si Si Si Si

Nota:

El elemento: <link>es un elemento vacio, contiene atributos solamente.

Este elemento va sólo en la seccion de cabeza,pero puede aparecer cualquier numero de veces.

Diferencias entre HTML 4.01 y HTML5.

Algunos atributos HTML 4.01 no son compatibles con HTML5

La etiqueta <link> no es soportada en HTML5.

Diferencias entre HTML y XHTML

En HTML la etiqueta <link> no tiene ninguna etiqueta final.

En XHTML la etiqueta <link> debe estar debidamente cerrado.
