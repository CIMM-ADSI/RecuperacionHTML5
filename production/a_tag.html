Definición y Uso.

La etiqueta <a> define un hipervínculo, que se utiliza para vincular de una página a otra.

El atributo más importante del elemento <a> es el atributo href, que indica el destino del vínculo.

Por defecto, los enlaces aparecerán de la siguiente manera en todos los navegadores:

- Un enlace no visitado está subrayado y azul
- Un enlace visitado es subrayada y púrpura
- Un vínculo activo es subrayada y rojo

Ajustes por defecto CSS

La mayoría de los navegadores mostrarán el elemento <a> con los siguientes valores predeterminados:

a:link, a:visited {
color: (internal value);
text-decoration: underline;
cursor: auto;
}
a:link:active, a:visited:active {
color: (internal value);
}

Diferencias entre HTML 4.01 y HTML5.

En HTML 4.01, la etiqueta <a> podría ser o bien un hipervínculo o un ancla. En HTML5, la etiqueta <a>es siempre un hipervínculo, pero si no tiene un atributo href, sólo es un marcador de posición para un hipervínculo.

HTML5 tiene algunos atributos nuevos, y algunos atributos HTML 4.01 ya no se soportan.
