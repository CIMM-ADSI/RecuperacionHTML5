Formulario <keygen>. Fecha: 12/05/2016

Un formulario con un campo keygen:

Ejemplo

<form action="demo_keygen.asp" method="get">
Username: <input type="text" name="usr_name">
Encryption: <keygen name="security">
<input type="submit">
</form>

Definición y Uso.

La etiqueta <keygen > especifica un campo generador de par de claves utilizado para las formas.

Cuando se envía el formulario, la clave privada se almacena localmente, y la clave pública se envía al servidor.

Soporte de Navegadores.

Los números de la tabla especifican la primera versión del navegador que es totalmente compatible con el elemento.

Diferencias entre HTML 4.01 y HTML5.

El elemento <keygen> es nuevo en HTML5.
