Titulo<title> Fecha: 13/05/2016

Definir un título para el documento HTML:

Ejemplo

<html>
<head>
<title>HTML Reference</title>
</head>

<body>
The content of the document......
</body>

</html>

Intentalo tu mismo »

Definición y Uso.

Se requiere de la etiqueta <title> en todos los documentos HTML y define el titulo del documento..


El elemento <title> :

  • define un titulo en la barra de herramientas del navegador.

  • proporciona el titulo de la pagina cuando se añade a los favoritos.

  • muestra el titulo de la pagina de resultados del motor de busqueda.

Soporte de Navegadores.

Elemento
<title> Si Si Si Si Si

Notas y sugerencias.

Nota: No se puede tener mas de un elemento <title> en un documento HTML.

Consejo: Si se omite la etiqueta <title>, el documento no se validara como HTML.
