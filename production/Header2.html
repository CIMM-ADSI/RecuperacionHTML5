Cabecera <header>. Fecha: 17/05/2016

Una cabecera para un <article>:

Ejemplo

<article>
<header>
  <h1>Most important heading here</h1>
  <h3>Less important heading here</h3>
  <p>Less important heading here</p>
</header>
  <p>Lorem Ipsum dolor set amet...</p>
</article>

Inténtalo tú mismo

Definición y Uso.

El elemento <header> representa un contenedor para el contenido de introducción o un conjunto de enlaces de navegación.

Un elemento <header> normalmente contiene:

  • uno o más elementos de encabezado (<h1> - <h6>)
  • logotipo o icono
  • información de autoría

Se pueden encontrar varios elementos <header> en un solo documento.

Nota:La etiqueta <header> no se puede colocar dentro de un <footer>, <address> u otro elemento <header>.

Diferencias entre HTML 4.01 y HTML5.

La etiqueta <header> es nueva en HTML5.

Atributos Globales

La etiqueta <header> es compatible con los atributos globales en HTML.

Atributos de eventos

La etiqueta <header> es compatible con los atributos de eventos en HTML.

Ajustes por defecto CSS

La mayoría de los navegadores mostrarán el elemento <header> con los siguientes valores predeterminados:

header {
display:block;
}
