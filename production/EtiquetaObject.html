Definición y Uso.

La etiqueta <object> define un objeto incrustado en un documento HTML. Utilice este elemento para incrustar multimedia (como audio, vídeo, applets de Java, ActiveX, PDF y Flash) en sus páginas web.

También puede utilizar la etiqueta <object> para insertar otra página web en el documento HTML.

Puede utilizar la etiqueta <param> para pasar parámetros a los plugins que se han incrustado con la etiqueta <object>.

Notas y sugerencias.

Nota: Un elemento <object> debe aparecer dentro del elemento <body>. El texto entre las etiquetas <object> y </object> es un texto alternativo, para los navegadores que no soportan esta etiqueta.

Consejo: Para imágenes utilizan la etiqueta <img> en lugar de la etiqueta <object>.

Consejo: Al menos uno de los "datos" o atributo "tipo" deberá ser definido.

Diferencias entre HTML 4.01 y HTML5.

Algunos atributos HTML 4.01 no son compatibles con HTML5.

El atributo "form" es nuevo en HTML5.

En HTML5, los objetos se pueden utilizar y presentarse en las formas.

En HTML5, los objetos ya no pueden aparecer dentro del elemento <head> de un documento.
