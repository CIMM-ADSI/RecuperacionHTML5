Cómo insertar una imagen:

Ejemplo

<img src="smiley.gif" alt="Smiley face" height="42" width="42">

Intenatalo tu mismo

Definición y Uso.

La etiqueta <img> define una imagen en una página HTML.

La etiqueta <img> tiene dos atributos requeridos: src y alt.

Nota:Las imágenes no se insertan técnicamente en una página HTML, las imágenes están vinculadas a las páginas HTML. La etiqueta <img> crea un espacio para la celebración de la imagen de referencia

Consejo:Para enlazar una imagen a otro documento, simplemente nido de la etiqueta <img> dentro <a>.

Diferencias entre HTML 4.01 y HTML5.

Los siguientes atributos: align, border, hspace, vspace y no son compatibles con HTML5.

Diferencias entre HTML y XHTML

En HTML de la etiqueta <img> no tiene ninguna etiqueta final.

En XHTML la etiqueta <img> debe estar debidamente cerrado.
