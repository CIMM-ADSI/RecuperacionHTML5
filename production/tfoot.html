tabla <tfoot> Fecha: 13/05/2016

Una tabla HTML con un <tfoot>, y un elemento <thead> <tbody>

Ejemplo

<table>
 <thead>
  <tr>
     <th>Month</th>
nbsp;     <th>Savings</th>
  </tr>
 </thead>
 <tfoot>
    <tr>
      <td>Sum</td>
      <td>$180</td>
    </tr>
 </tfoot>
 <tbody>
    <tr>
      <td>January</td>
    <td>$100</td>
    </tr>
    <tr>
      <td>February</td>
      <td>$80</td>
  </tr>
 </tbody>
</table>

Definición y Uso.

La etiqueta <tfoot> se utiliza en conjunción con los<tfoot>

El elemento <thead>and <tbody> elementos para especificar cada parte de una tabla (pie de pagina, erncabezado, cuerpo).

Los navegadores pueden utilizar estos elementos para permitir el desplazamiento del cuerpo de la tabla de forma independiente de la cabecera y el pie. Además, cuando se imprime una gran tabla que abarca varias páginas, estos elementos pueden permitir a la cabecera de la tabla y el pie de página que se imprimirá en la parte superior e inferior de cada página.

la etiqueta <tfoot>debe ser utilizada en el siguiente contexto: como hijo de un elemento<table> despues de cualquier <caption> , <colgroup> y <thead>elementos y antes de cualquier etiqueta <tbody> y <tr> elementos

Notas y sugerencias.

Nota : El elemento <tfoot> debe tener una o mas etiquetas <tr> interior.


Consejo: La <thead> ,<tbody> y <tfoot> elementos que no modifiquen la composicion de la tabla por defecto. Sin embargo, puede utilizar CSS para el estilo de estos elementos.

Diferencias entre HTML 4.01 y HTML5.

Ninguno de los atributos HTML 4.01 son compatibles con HTML5.

Atributos

Atributo Valor Descripcion
align right left center justify char No soportado en HTML5.
 Alinear el contenido dentro del elemento <tfoot>
Char Carácter No soportado en HTML5.
 Alinear el contenido dentro del elemento <tfoot> a un carácter
charoff Numero No soportado en HTML5.
 Establece el número de caracteres del contenido dentro del elemento <tfoot> se alineará desde el carácter especificado por el atributo Char
Valing Top middle bottom baseline No soportado por HTML5.
 Alinear verticalmente el contendo dentro del elemento <tfoot>
