Definición y Uso.

La etiqueta <i> define una parte del texto en un estado de ánimo de voz o alternativo. El contenido de la etiqueta <i> por lo general se muestra en cursiva.

La etiqueta <i> se puede utilizar para indicar un término técnico, una frase de otra lengua, un pensamiento o un nombre de enviar, etc.

Utilice el <i> elemento sólo cuando no hay un elemento semántico más apropiado, tales como:

  • <em>(texto subrayado)
  • <strong>(texto importante)
  • <mark>(marca / texto resaltado)
  • <cite>(El título de un trabajo)
  • <dfn>(término de definición)

Diferencias entre HTML 4.01 y HTML5.

En HTML5,se utiliza la etiqueta <i> para representar el texto en cursiva. Sin embargo, esto no es necesariamente con HTML5. Las hojas de estilo se pueden utilizar para dar formato al texto dentro del elemento <i>.
