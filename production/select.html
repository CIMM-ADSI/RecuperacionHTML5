    • Settings 1

Seleccionar <select>. Fecha: 17/05/2016

Ejemplo

Crear una lista desplegable con cuatro opciones:

<select>
<option value="volvo">Volvo</option>
<option value="saab">Saab</option>
<option value="mercedes">Mercedes</option>
<option value="audi">Audi</option>
</select>

Definición y Uso.

El elemento <select> se utiliza para crear una lista desplegable.

Las <option> etiquetas dentro del elemento <select> definen las opciones disponibles en la lista.

Soporte de Navegadores.

Elemento
<select> Si Si Si Si Si

Notas y Sugerencias.

Consejo: El elemento <select> es un control de formulario y se puede utilizar en un formulario para recolectar información del usuario.

Diferencias entre HTML 4.01 y HTML5.

HTML5 ha añadido algunos nuevos atributos.
