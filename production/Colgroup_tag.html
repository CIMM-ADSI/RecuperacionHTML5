Definición y Uso.

La etiqueta <colgroup> especifica un grupo de una o más columnas en una tabla para el formato.

La etiqueta <colgroup>es útil para aplicar estilos de columnas completas, en lugar de repetir los estilos para cada celda, para cada fila.

NotaLa etiqueta <colgroup> debe ser un hijo de un elemento <table> , después de cualquier <caption> elementos y antes de cualquier <culata en T>, <tbody>, <tfoot> y <tr> elementos..

ConsejoPara definir propiedades diferentes a una columna dentro de un <colgroup>, utilice el <col> etiqueta dentro de la etiqueta <colgroup>.
