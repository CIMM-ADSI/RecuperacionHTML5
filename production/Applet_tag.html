    • Settings 1

Ejemplo <applet>. Fecha: 18/05/2016

Un applet de Java embebido:

<applet code="Bubbles.class" width="350" height="350">
Java applet that draws animated bubbles.
</applet>

Definición y Uso.

La etiqueta <applet> no es compatible con HTML5. utilice <embed> o <object> en su lugar.

La etiqueta <applet> define un applet incrustado.

Soporte de Navegadores.

Elemento
<applet> No soportado No soportado Si Si No soportado

Nota:Todavía hay cierto apoyo a la etiqueta <applet> en algunos navegadores, pero requiere plug-ins / instalaciones adicionales para trabajar.

Nota:La etiqueta <applet> es compatible con Internet Explorer 11 y versiones anteriores, el uso de un plug-in.

los Atributos requeridos

Atributo Valor Descripción
code URL Especifica el nombre de archivo de un applet de Java
object name Especifica una referencia a una representación serializada de un applet
