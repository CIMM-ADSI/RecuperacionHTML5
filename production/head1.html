Cabecera <head>. Fecha: 17/05/2016

Un documento HTML, con una etiqueta <title> dentro de la sección de la cabecera:

Ejemplo

<!DOCTYPE html>
<html>
<head>
<title>Titulo del documento</title>
</head>
   
<body>
El contenido del documento....
</body>
   
</html>

Definición y Uso.

El elemento <head> es un contenedor para todos los elementos de cabecera.

El elemento <head> puede incluir un título para el documento, scripts, estilos, meta information, y mucho más.

Los siguientes elementos pueden ir dentro del elemento <head>:

  • <title>(se requiere este elemento en un documento HTML)
  • <style>
  • <base>
  • <link>
  • <meta>
  • <script>
  • <noscript>

Soporte de Navegadores.

Elemento

Diferencias entre HTML 4.01 y HTML5.

En HTML 4.01 el elemento <head> se requiere.

En HTML 5, el elemento <head> se puede omitir. El siguiente código validará como HTML5:

Ejemplo

<!DOCTYPE html>
<html>
<title>Titulo del documento</title>
   
<body>
<h1>Esto es un titulo</h1>
<p>Esto es un parrafo</p>
</body>
   
</html>
