                            <form action="demo_form.asp" method="get">

Ver Ejemplo

Definición y Uso.

La etiqueta <form> se utiliza para crear un formulario HTML para la entrada del usuario.

El elemento <form> puede contener uno o más de los siguientes elementos de formulario:

  • <Input>
  • <Textarea>
  • <Botón>
  • <Select>
  • <Opción>
  • <Optgroup>
  • <Fieldset>
  • <Label>

Diferencias entre HTML y XHTML

HTML5 ha añadido dos nuevos atributos: autocompletar y novalidate, y se retira el atributo aceptar.

Diferencias entre HTML 4.01 y HTML5.

En XHTML, el atributo de nombre está en desuso. Utilice el atributo id mundial en su lugar.

Atributos.

Atributo Valor Descripcion
accept file_type No se admite en HTML5 .
Especifica una lista separada por comas de los tipos de archivos que acepta el servidor ( que se pueden enviar a través de la carga de archivos )
