h1 a h6 < Etiquetas>. Fecha: 18/05/2016

Los seis diferentes HTML epígrafes:

Ejemplo
 
                           <h1>This is heading 1</h1> 

                           <h2>This is heading 2</h2> 

                           <h3>This is heading 3</h3> 

                           <h4>This is heading 4</h4> 
 
                           <h5>This is heading 5</h5> 

                           <h6>This is heading 6</h6>

Ver Ejemplo

Definición y Uso.

La <h1> a <h6> etiquetas se utilizan para definir HTML partidas.

<H1> define el título más importante. <H6> define el menos partida importante.

Diferencias entre HTML 4.01 y HTML5.

El atributo "align" no es compatible con HTML5. Usar CSS para alinear elementos.

Atributos .

Atributo Valor Descripcion
align left
center
right
justify		 No se admite en HTML5 .
Especifica la alineación de un encabezamiento

Notas de compatibilidad.

El atributo de alineación de <h1> a <h6> no es compatible con HTML5. Usar CSS en su lugar.

Sintaxis CSS: <h1 estilo = "text-align: right">

CSS Ejemplo: Establecer la alineación de las partidas

En nuestro tutorial CSS se pueden encontrar más detalles acerca de la propiedad text-align .

Sintaxis 

                                  <h1 align="left|right|center|justify">

Los atributos globales.

La <h1> a <h6> etiquetas también es compatible con el de los atributos globales en HTML .

Atributos de eventos

La <h1> a <h6> etiquetas también es compatible con atributos de eventos en HTML .
