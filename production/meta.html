Etiqueta <meta>. Fecha: 12/05/2016

Ejemplo

Describir los metadatos dentro de un fdocumento HTML:

<head>
<meta charset="UTF-8">
meta name="description" content="Free Web tutorials">
<meta name="Keywords" content="Hege Refsnes">
</head>

Definición y Uso.

Los metadatos son (informacion) sobre los datos.

La etiqueta <meta> proporciona metadatos sobre el documento HTML. Los metadatos no se mostrarán en la página, pero estará apta para su procedimiento.

Los metaelementos se utilizan normalmente para especificar descripción de páginas, palabras clave, autor del documento, modificada por última vez, y otros metadatos.

Los metadatos pueden ser utilizados por los navegadores (cómo mostrar contenido o recarga de la pagina), los motores de búsqueda (palabras clave), u otros servicios web.

Soporte de Navegadores.

Elemento
<meta> Si Si Si Si Si

Nota:Las etiquetas <meta>siempre van dentro del elemento <head>

Nota:los metadatos siempre pasan como nombre/valor pares.

Nota:El atributo de contenido deberá ser definido si se define el nombre o el atributo http-equiv. Si no se define ninguna de ellas, no se puede definir el atributo de contenido.

Diferencias entre HTML 4.01 y HTML5.

El atributo de esquema no es compatible con HTML5.

HTML5 tiene un nuevo atributo, charset, que hace más facil para definir charset:

- HTML 4.01: <meta http-equiv="Content-Type" content="text/html; charset = UTF-8">

-HTML5: <charset meta = "UTF-8">

Diferencias entre HTML 4.01 y XHTML.

En HTML la etiqueta <meta> no tiene ninguna etiqueta final.

En XHTML la etiqueta <meta> debe estar debidamente cerrado.

Ejemlos

Ejemplo 1-Definición de palabras clave para los motores de búsqueda:

<meta name="Keywords" content="HTML, CSS, XML, XHTML, JavaScript">

Ejemplo 2-Definir una descripción de su página web:

<meta name="description" content="Free Web tutorials on HTML and CSS">

Ejemplo 3-Definir el autor de una página web:

<meta name="author" content="Hege Refsnes">

Ejemplo 4-Actualizar el documento cada 30 segundos:

<meta http-equiv="refresh" content="30">
