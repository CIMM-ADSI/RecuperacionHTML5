Formulario <input>. Fecha: 12/05/2016

Un formulario HTML con tres campos de entrada; dos campos de texto y un botón de envío:

Ejemplo

<form action="demo_form.asp">
First name: <input type="text" name="fname"><br>
Last name: <input type="text" name="lname"><br>
<input type="submit" value="Submit">
</form>

Definición y Uso.

La etiqueta <input> especifica un campo de entrada donde el usuario puede introducir datos.

<Input> se utilizan dentro de un elemento <form> para declarar controles de entrada que permiten a los usuarios a los datos de entrada.

Un campo de entrada puede variar de muchas maneras, dependiendo del tipo de atributo.

Notas y sugerencias

Nota: El elemento <input> está vacía, contiene atributos solamente.

Consejo:Utilice la etiqueta <label> elemento para definir etiquetas para los elementos <input>.

Diferencias entre HTML 4.01 y HTML5.

El atributo " align" no es compatible con HTML5.

En HTML5, la etiqueta <input> tiene varios atributos nuevos, y el atributo type tiene varios valores nuevos.

Diferencias entre HTML y XHTML.

En HTML, la etiqueta <input> no tiene ninguna etiqueta final.

En XHTML, la etiqueta <input> debe estar debidamente cerrado, como este <input />.
