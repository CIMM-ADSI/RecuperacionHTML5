Font <Font>. Fecha: 12/05/2016

Especificar el tamaño de fuente, tipo de letra y el color del texto:

Ejemplo

<font size="3" color="red">This is some text!</font> <font size="2"
color="blue">This is some text!</font> <font face="verdana"
color="blue">This is some text!</font> <font face="verdana"

Definición y Uso.

La etiqueta <Font> no es compatible con HTML5. Usar CSS en su lugar

La etiqueta <Font> especifica el tipo de letra, tamaño de fuente, y el color del texto.
