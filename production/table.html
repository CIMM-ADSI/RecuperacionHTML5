HTML <table> Tag. Fecha: 12/05/2016

Ejemplo

Una tabla HTML sencilla, que contiene dos columnas y dos filas:

  <table>

    <tr>

      <th>Month</th>

      <th>Savings</th>
 
     </tr> 

     <tr> 

      <td>January</td> 

      <td>$100</td> 

    </tr> 

  </table>

Definición y Uso

La etiqueta <table> define una tabla HTML.
Una tabla HTML consiste en el elemento <table> y uno o más <tr> , <th> y <td> elementos.
La <tr> define una fila de la tabla, el <th> define un encabezado de la tabla, y el <td> define una celda de tabla.
Una tabla HTML más complejo también puede incluir <caption>, <col>, <colgroup>, <culata en T>, <tfoot> y <tbody> elementos.
