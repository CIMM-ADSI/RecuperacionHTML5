HTML <td> Tag. Fecha: 12/05/2016

Ejemplo

Una tabla HTML sencilla, con dos celdas de la tabla:

                            
  <table>

    <tr>

      <td>Cell A</td>

      <td>Cell B</td>

    </tr>

  </table>

Definición y Uso

La etiqueta <td> define un estándar de celda en una tabla HTML.

Una tabla HTML tiene dos tipos de células:

- Celdas de encabezado contiene información de cabecera (creada con el <th> elemento)

- células estándar contiene los datos (creada con el elemento <td>)

El texto en <th> elementos están en negrita y centrado de forma predeterminada.

El texto en los elementos <td> son regulares y por defecto alineado a la izquierda.

    display: table-cell;
