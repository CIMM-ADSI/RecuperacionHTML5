Botones <label>. Fecha: 12/05/2016

Tres botones de radio con las etiquetas:

Ejemplo

<form action="demo_form.asp">
<label for="male">Male</label>
<input type="radio" name="gender" id="male" value="male"><br>
<label for="female">Female</label>
<input type="radio" name="gender" id="female" value="female"><br>
<label for="other">Other</label>
<input type="radio" name="gender" id="other" value="other"><br>
<br> <input type="submit" value="Submit">
</form>

Definición y Uso.

La etiqueta <label>define una etiqueta para un elemento <imput>.

El elemento <label> no hace como algo especial para el usuario. Sin embargo, proporciona una mejora usabilidad para los usuarios de ratón, ya que si el usuario hace clic en el texto dentro del elemento <label>, se activa o desactiva el control.

El atributo de la etiqueta <label> debe ser igual al atributo id del elemento relacionado para unirlas.

Notas y sugerencias.

Consejo: Un marcador se puede unir a un elemento ya sea usando el atributo "for", o colocando el elemento dentro del elemento <label>.

Diferencias entre HTML 4.01 y HTML5.

El atributo "forma" es nuevo en HTML5.

atributos

     = Nuevo en HTML5

Atributo Valor Descripción
for      element_id Especifica qué elemento forma una etiqueta está obligado a
form      form_id Especifica una o más formas la etiqueta pertenece a
