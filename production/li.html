Lista <li>. Fecha: 12/05/2016

Una ordenada (<ol>) y uno no ordenada (<ul>) Lista HTML:

Ejemplo

<ol>
<li>Coffee</li>
<li>Tea</li>
<li>Milk</li> </ol>

<ul>
<li>Coffee</li>
<li>Tea</li>
<li>Milk</li>
</ul>

Definición y Uso.

La etiqueta <li< define un elemento de la lista.

La etiqueta <li< se utiliza en las listas ordenadas ( <ol> ), listas desordenadas ( <ul> )>, y en las listas de menú ( <Menú> ).

Soporte de Navegadores.

Elemento

Diferencias entre HTML 4.01 y HTML5.

El atributo "tipo" no es compatible con HTML5.

El atributo "value" fue desaprobado en HTML 4.01, pero es compatible con HTML5
