Strong <strong>. Fecha: 19/05/2016

Ejemplo

Dar formato al texto en un documento:

<strong>Strong text</strong>

Definición y Uso.

La etiqueta <strong> es una etiqueta frase. Definir el texto importante.

Consejo: esta etiqueta no está en desuso, pero es posible lograr un efecto más rico con CSS.

Todas las etiquetas de la frase:

Etiqueta Descripción
<em> Texto subrayado
<strong> Define texto importante
<code> Define una pieza de código informático
<samp> Define un ejemplo de salida de un programa de ordenador
<kbd> Define la entrada de teclado
<var> Define una variable

Soporte de Navegadores.

Elemento
<strong> Si Si Si Si Si

Diferencias entre HTML 4.01 y HTML5.

En HTML 4.01, la etiqueta <strong> define el texto con énfasis fuerte, pero en HTML5 define texto importante.
