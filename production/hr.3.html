<hr>. Fecha: 17/05/2016

Se utiliza la etiqueta <hr> para definir un cambio en el contenido temático:

Ejemplo

<h1>HTML</h1>
<p>HTML es un lenguaje para la descripcion de paginas web...</p>
   
<hr>
   
<h1>CSS</h1>
<p>CSS define cómo mostrar los elementos HTML...</p>

Definición y Uso.

La etiqueta <hr> define una ruptura temática en una página HTML (por ejemplo, un cambio de tema).

El elemento <hr> se utiliza para separar el contenido (o definir un cambio) en una página HTML.

Soporte de Navegadores.

Elemento
<header> Si Si Si Si Si

Diferencias entre HTML 4.01 y HTML5.

En HTML5, la etiqueta <hr> define una ruptura temática.

En HTML 4.01, la etiqueta <hr> representa una linea horizontal.

Sin embargo, la etiqueta <hr> todavía se puede mostrar como una línea horizontal en los navegadores visuales, pero ahora se define en términos semánticos, en lugar de términos de presentación.

Todos los atributos de diseño se eliminan en HTML5. Usar CSS en su lugar.

Atributos Globales

La etiqueta <hr> es compatible con los atributos globales en HTML.

Atributos de eventos

La etiqueta <hr> es compatible con los atributos de eventos en HTML.

Ajustes por defecto CSS

La mayoría de los navegadores mostrarán el elemento <hr> con los siguientes valores predeterminados:

Ejemplo
hr {
   display: block;
   margin-top: 0.5em;
   margin-bottom: 0.5em;
   margin-left: auto;
   margin-right: auto;
   border-style: inset;
   border-width: 1px;
}
