Teletipo <noframes>. Fecha: 16/05/2016

Ejemplo

<html>

<frameset cols="25%,50%,25%">
<frame src="frame_a.htm">

<noframes>Sorry, your browser does not handle frames!</noframes>
</frameset>

Inténtalo tú Mismo

Definición y Uso.

La etiqueta <noframes> no es compatible con HTML5.


La etiqueta <noframes> es una etiqueta de retorno para los navegadores que no soporten marcos. Puede contener todos los elementos HTML que se pueden encontrar en el interior del elemento < body > de una página HTML normal.


El elemento <noframes> se puede utilizar para vincular a una versión no-marcos de la página web o para mostrar un mensaje a los usuarios que se requieren marcos.


El elemento <noframes> va dentro del elemento <frameset>.


Nota: Si desea validar una página que contiene marcos, asegúrese de que el <!DOCTYPE> se ha configurado como "HTML Frameset DTD" o "XHTML Frameset DTD".

Diferencias entre HTML y XHTML.

Importante: En XHTML DTD conjunto de marcos, el texto en el elemento <noframes> debe estar encerrado en un elemento <body>.
