<iframe> Fecha: 12/05/2016

Un marco en línea está marcado de la siguiente manera:

Ejemplo

<iframe src="http://www.w3schools.com"> </iframe>

Intenatalo tu mismo

Definición y Uso.

La etiqueta <iframe> especifica un marco en línea.

LUn marco en línea se utiliza para insertar otro documento dentro del documento HTML actual.

Soporte de Navegadores.

Elemento
<iframe> Si Si Si Si Si

Notas y sugerencias.

Consejo:Frente a los navegadores que no soportan <iframe>, agregue un texto entre la etiqueta de apertura <iframe> y el cierre </ iframe>.

Consejo:Usar CSS para estilizar el <iframe> (incluso para incluir barras de desplazamiento).

Diferencias entre HTML 4.01 y HTML5.

HTML5 ha añadido algunos nuevos atributos, y varios atributos HTML 4.01 se eliminan de HTML5.

Diferencias entre HTML y XHTML

En XHTML, el atributo de nombre está en desuso, y se eliminará. Utilice el atributo id mundial en su lugar.

Ajustes por defecto CSS.

La mayoría de los navegadores mostrarán el elemento <iframe> con los siguientes valores predeterminados:

iframe:focus {
  outline: none;
}
 
iframe[seamless] {
  display: block;
}
