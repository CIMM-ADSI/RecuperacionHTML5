Grupo de Opciones <optgroup>. Fecha: 17/05/2016

Ejemplo

Grupo de opciones con < > optgroup etiquetas relacionadas:

<select>
<optgroup label="Swedish Cars">
<option value="volvo">Volvo</option>
<option value="saab">Saab</option>
</optgroup>
<optgroup label="German Cars">
<option value="mercedes">Mercedes</option>
<option value="audi">Audi</option>
</optgroup>
</select>


Inténtalo tú mismo

Definición y Uso.

El <optgroup> se utiliza para las opciones relacionadas con los grupos en una lista desplegable.

Si usted tiene una larga lista de opciones, grupos de opciones relacionadas son más fáciles de manejar para un usuario.

Soporte de Navegadores.

Elemento

Diferencias entre HTML 4.01 y HTML5.

Atributos.

Atributo Valor Descripción
disabled disabled Especifica que un grupo de opción se desactivará.
label text Especifica una etiqueta para una opción de grupo.

Atributos Globales.

La etiqueta <optgroup> también es compatible con los atributos globales en HTML.

Atributos de Eventos.

La etiqueta <optgroup> también apoya los atributos de eventos en HTML.
