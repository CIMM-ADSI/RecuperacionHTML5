Definición y Uso.

La etiqueta <frame> no es compatible con HTML5.

La etiqueta <frame> define una ventana en particular ( estructura ) dentro de un <frame>

Cada <frame> en un <frameset> puede tener diferentes atributos, como la frontera, el desplazamiento, la capacidad de cambiar el tamaño, etc.

Nota: Si desea validar una página que contiene marcos, asegúrese de que el <! DOCTYPE> se ha configurado como "conjunto de marcos HTML DTD" o "conjunto de marcos XHTML DTD".

Diferencias entre HTML y XHTML

En HTML, la etiqueta <frame> no tiene ninguna etiqueta final. En XHTML, la etiqueta <frame> debe estar debidamente cerrado.

Diferencias entre HTML 4.01 y HTML5.

La etiqueta <frame> no es compatible con HTML5.
