Suprimir <ins>. Fecha: 12/05/2016

Un texto con una parte suprimida, y una nueva pieza insertada:

Ejemplo

<p>My favorite color is <del>blue</del> <ins>red</ins>!</p>

Definición y Uso.

La etiqueta <ins> define un texto que se ha insertado en un documento.

Consejo: También busque en el <del> etiqueta para el marcado de texto eliminado.

Los navegadores normalmente golpearán a través de una línea de texto borrada y subrayar el texto insertado.

Soporte de Navegadores.

Elemento
<ins> Si Si Si Si Si

Notas y sugerencias.

Consejo: Utilice <ins> junto con <Del> para el marcado de cambios y modificaciones en un documento.

Diferencias entre HTML 4.01 y HTML5.

NINGUNA

atributos

Atributo Valor Descripción
cite URL Especifica una dirección URL a un documento que explica la razón por la cual se inserta el texto / cambiado
datatime YYYY-MM-DDThh:mm:ssTZD Especifica la fecha y la hora en que se inserta el texto / cambiado

Los atributos globales.

La etiqueta <ins> también es compatible con el de los atributos globales en HTML .

Atributos de eventos.

La etiqueta <ins> también apoya la atributos de eventos en HTML .

Ajustes por defecto CSS.

La mayoría de los navegadores mostrarán el elemento <ins>con los siguientes valores predeterminados:

Ejemplo

ins {
text-decoration: underline;
}
