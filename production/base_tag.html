Base <base>. Fecha: 17/05/2016

Ejemplo

Especificar una dirección URL predeterminada y un objetivo predeterminado para todos los enlaces en una página :

<head>
<base href="http://www.w3schools.com/images/" target="_blank">
</head>

<body>
<img src="stickman.gif" width="24" height="39" alt="Stickman">
<a href="http://www.w3schools.com">W3Schools</a>
</body>

Definici√≥n y Uso.

La etiqueta <base> especifica la base URL / objetivo para todas las URL relativas en un documento.

Puede haber como máximo un elemento <base> en un documento , y debe estar dentro del elemento < head>.

Soporte de Navegadores.

Elemento
<base> Si Si Si Si Si

Notas y sugerencias

Consejo: Ponga la etiqueta <base> como el primer elemento dentro del elemento < head> , de modo que otros elementos en la sección de cabecera utiliza la información del elemento <base >.

Nota: Si la etiqueta <base > está presente , debe tener ya sea un atributo href o un atributo de destino , o ambas cosas.

Diferencias entre HTML 4.01 y HTML5

Ninguna

Diferencias entre HTML y XHTML

En HTML la etiqueta <base> no tiene ninguna etiqueta final.

En XHTML la etiqueta <base > debe estar debidamente cerrado.
