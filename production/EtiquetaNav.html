Objeto <nav>. Fecha: 16/05/2016

Ejemplo

Un conjunto de enlaces de navegación:

<nav>
<a href="/html/">HTML</a> |
<a href="/css/">CSS</a> |
<a href="/js/">JavaScript</a> |
<a href="/jquery/">jQuery</a>
</nav>

Inténtalo tú mismo

Definición y Uso.

La etiqueta <nav> define un conjunto de enlaces de navegación.

Tenga en cuenta que no todos los enlaces de un documento deben estar dentro de un elemento <nav>. El elemento <nav> se dirige a los principales bloques de vínculos de navegación.

Navegadores, tales como lectores de pantalla para usuarios con discapacidad, pueden utilizar este elemento para determinar si se debe omitir la prestación inicial de este contenido.

Diferencias entre HTML 4.01 y HTML5.

La etiqueta <nav> es nueva en HTML5.

Atributos Globales.

La etiqueta <nav> también soporta los atributos globales en HTML.

Atributos de eventos.

La etiqueta <nav> también soporta los atributos del evento en HTML.

Ajustes por defecto CSS.

La mayoría de los navegadores mostrarán el elemento <nav> con los siguientes valores predeterminados:

nav {
display: block;
}
