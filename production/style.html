Estilo <style>. Fecha: 24/05/2016

Ejemplo

El uso del elemento <style> en un documento HTML:

<html>
<head>
<style>
h1 {color:red;}
p {color:blue;}
</style>
</head>
<body>


<h1>A heading</h1>
<p>A paragraph.</p>


</body>
</html>

Definición y Uso.

La etiqueta <style> se utiliza para definir la información de estilo para un documento HTML.

Dentro del elemento <style> especifica cómo los elementos HTML deben representar en un explorador.

Cada documento HTML puede contener múltiples <style>.

Notas y Sugerencias.

Consejo: Para enlazar a una hoja de estilos externa, utilice el <link> etiqueta.

Consejo: Para obtener más información sobre las hojas de estilo, por favor lea nuestra Tutorial CSS .

Nota: Si el "con ámbito" atributo no se utiliza, cada etiqueta <style> debe estar ubicada en la sección de cabeza.

Diferencias entre HTML 4.01 y HTML5.

El atributo "con ámbito" es nuevo en HTML5, que permite definir los estilos de una sección específica del documento. Si el atributo "con ámbito" está presente, los estilos sólo se aplican al elemento padre del elemento de estilo y elementos secundarios de ese elemento.
