Lista Ordenada <ol>. Fecha: 17/05/2016

Ejemplo

2 listas diferentes ordenadas:

<ol>
<li> Café </li>
<li> té </li>
<li> Leche </li>
</ol>

<ol start = "50">
<li> Café </li>
<li> té </li>
<li> Leche </li>
</Ol>

Definición y Uso.

La etiqueta <ol> define una lista ordenada. Una lista ordenada puede ser numérico o alfabético.

Utilice el <li> etiqueta para definir elementos de la lista.

Soporte de Navegadores.

Elemento

Notas y sugerencias.

Consejo: Para la lista no ordenada, utilice la etiqueta <ul>.

Consejo: Usar CSS para Estilo listas.

Diferencias entre HTML 4.01 y HTML5.

Los atributos "start" y "type" fueron desaprobados en HTML 4.01, pero son compatibles con HTML5.

El atributo "reverset" es nuevo en HTML5.

El atributo "compact" no es compatible con HTML5.

Ajustes por defecto CSS.

La mayoría de los navegadores mostrarán el elemento <ol> con los siguientes valores predeterminados:

ol {
display: block;
list-style-type: decimal;
margin-top: 1em;
margin-bottom: 1em;
margin-left: 0;
margin-right: 0;
padding-left: 40px;
}
