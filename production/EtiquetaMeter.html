Objeto medidor <meter>. Fecha: 13/05/2016

Ejemplo

Usa el elemento meter para medir los datos dentro de un rango dado (un medidor):

<meter value="2" min="0" max="10"> 2 de cada 10 </meter><br>
<meter value="0.6"> 60% </meter>

Inténtalo tú mismo

Definición y Uso.

El <meter> define una medida escalar dentro de un rango conocido, o un valor fraccionario. Esto también se conoce como un indicador.

Ejemplos: Uso del disco, la relevancia de un resultado de la consulta, etc.

Nota: La etiqueta <meter> no debe utilizarse para indicar el progreso (como en una barra de progreso). Para barras de progreso, utilice la etiqueta <progress>.

La etiqueta <meter> es nueva en HTML5.

Atributos.

Atributo Valor Descripción

high number Especifica el rango que se considera que es un valor alto
low number Especifica el rango que se considera que es un valor bajo
max number Especifica el valor máximo del rango
min number Especifica el valor mínimo del rango
optimum number Especifica qué valor es el valor óptimo para el indicador
value number Requiere. Especificar el valor actual del medidor
