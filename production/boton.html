define un botón que se pulse.<button>. Fecha: 18/05/2016

Ejemplo

Un botón puede hacer clic está marcado como sigue:

<button type="button">Click Me!</button>

INTENTALO TU MISMO .

Definición y Uso.

El botón <> define un botón que se pulse.

Dentro de un elemento <button> se puede poner el contenido, como texto o imágenes. Esta es la diferencia entre este
elemento y botones creados con el elemento <input>.

Consejo: Siempre especifique el atributo de tipo para un elemento <botón>. Los distintos navegadores utilizan
diferentes tipos predeterminados para el elemento <botón>.

Notas y sugerencias.

Nota: Si se utiliza el botón <> en un formulario HTML, diferentes navegadores pueden presentar valores diferentes. Utilice <input> para crear botones en un formulario HTML.

Consejo: Visita nuestra CSS Botones Tutorial para encontrar la manera de botones de estilo con CSS.

Diferencias entre HTML 4.01 y HTML5.

HTML5 tiene los siguientes atributos nuevos: enfoque automático, la forma, formAction, formenctype, formmethod, formnovalidate, y formtarget.
